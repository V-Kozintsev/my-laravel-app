**Какие команды запускать после клонирования**

- Установить PHP-зависимости:

```
composer install
```

- Установить Node.js-пакеты:

```
npm install
```

- Создать файл окружения `.env` (если он отсутствует):

- Сгенерировать ключ приложения:

```
php artisan key:generate
```

- Выполняем миграции:

``` 
php artisan migrate
```
